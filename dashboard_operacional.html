⚙ Configuração — GAS Proxy

Este dashboard consome dados via Google Apps Script publicado como Web App, autenticado pela conta do proprietário do script — sem expor credenciais no browser.

Como configurar o Apps Script:
1. Acesse script.google.com → Novo projeto
2. Cole o conteúdo do arquivo Code.gs fornecido
3. Edite SHEET_ID e SHEET_NAME no topo
4. Implantar → Nova implantação → Web App
5. Executar como: Eu (sua conta Google)
6. Quem tem acesso: Qualquer pessoa (ou seu domínio)
7. Copie a URL gerada e cole abaixo
URL gerada após publicar. Deve terminar em /exec
